# Skill test for PHP, XML and SQL

## Prerequisites

Make sure you have the following installed on your system:

- Docker
- Docker Compose
- Git

## Installation

Follow these steps to setup the project:

1. **Clone the repository**

    ```bash
    git clone [https://github.com/yourusername/cyberplatform.git](https://github.com/your-username/phpXmlSql.git)
    cd phpXmlSql
    ```

2. **Build the Docker images**

    ```bash
    docker-compose build
    ```

3. **Run the Docker containers**

    ```bash
    docker-compose up -d
    ```

## Usage

You can access the application in your web browser at `http://localhost`.

To stop the application, use the following command:

```bash
docker-compose down
